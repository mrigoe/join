Get involved

You can get involved with the Minetest community in several ways:

  • Giving support on IRC
  • Giving support on the forums, publishing mods, subgames, maps, texture packs
  • Contribute on GitHub - report bugs, make pull requests
  • Talk about Minetest on reddit
  • Perform some development
