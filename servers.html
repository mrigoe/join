Flags

Flag Explanation
Pwd Players need to set a password at first connect
Dmg Damage enabled
Pvp Players can damage other players
Ded Dedicated Server
Liq Finite Liquids
Rol Rollback enabled
Cre Creative mode
Far Players can see far away names

Announcing Servers

Set this in your server configuration: 

server_announce = true
server_name = My server
server_description = A server that I made.
server_url = <URL to server webpage or forum topic>
